# doppelganger

Clone your friend's voice and use them as a Discord bot

## Development

### Training

- Record 3-5 minutes of voice with minimal noise and place in `training-data/${DISCORD_USER}/sample.wav`
  - clean up noise with Audacity (Effect > Noise Removal and Repair > Noise Reduction)
